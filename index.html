About this data

Figures are parsed from the SEC XBRL company concept endpoint for dei:EntityCommonStockSharesOutstanding. We filter for numeric values with fiscal years strictly greater than 2020, then show the highest and lowest values observed.

By default, the page renders from the bundled data.json (Biogen). If a ?CIK= query parameter is present, the app fetches the live SEC API via a CORS-friendly proxy and updates the page dynamically.

Built for GitHub Pages. Data courtesy of the U.S. SEC. This is a community demo and not investment advice.
